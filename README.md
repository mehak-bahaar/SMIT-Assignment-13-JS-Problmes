# JavaScript Problems - Assignment 13

This project contains various JavaScript calculators and simulations demonstrating fundamental programming concepts.

## Features

### 1. Grocery Bill Calculator
- Calculates total bill for multiple grocery items
- Applies 10% discount for bills over $5000

### 2. Student Grade Calculator
- Calculates average grade from 5 subject marks
- Assigns letter grades (A, B, C, F) based on average

### 3. ATM Withdrawal Simulation
- Simulates ATM withdrawal with balance checking
- Prevents withdrawal if insufficient funds

### 4. Restaurant Order System
- Orders from menu items (Burger, Pizza, Soda)
- Applies 15% discount for orders over Rs. 2000

### 5. Employee Salary Calculator
- Calculates weekly salary based on hours worked
- Includes overtime pay (1.5x rate) for hours over 40

## Technologies Used
- HTML5
- JavaScript (ES6+)

## How to Run
1. Open `index.html` in any modern web browser
2. The JavaScript will execute automatically and prompt for inputs through dialog boxes
3. Follow the prompts to interact with each calculator/simulation

## Project Structure
- `index.html` - Main HTML file that loads the JavaScript
- `index.js` - Contains all the calculator and simulation functions
